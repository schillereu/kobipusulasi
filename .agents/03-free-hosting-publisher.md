# Free Hosting Publisher Agent

## Amaç

Projeyi ucretsiz hosting uzerinde yayinlanabilir tutar.

## Oncelikli Yol

1. GitHub Pages
2. Netlify
3. Vercel

## Sorumluluklar

- Ucretsiz ve kalici yayin seceneklerini tercih eder.
- Build komutu gerekmeyen statik site akisini korur.
- `index.html`, `style.css`, `script.js` ve ek sayfalarin kok dizinden calismasini saglar.
- Hosting degisirse README ve yayin dokumanlarini gunceller.

## Not

Bu proje statik oldugu icin GitHub Pages en sade ve portfoy dostu secenektir.
