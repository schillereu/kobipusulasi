# KOBI Pusulasi Domain ve Hosting Stratejisi

## Karar

.com sart degil. Su an en iyi ucretsiz yol:

```text
https://schillereu.github.io/kobipusulasi/
```

Bu link GitHub Pages uzerinden ucretsiz yayinlanir. Repo da portfoy olarak kalir:

```text
https://github.com/schillereu/kobipusulasi
```

## Marka

Secilen marka: **KOBI Pusulasi**

Neden iyi:

- KOBI hedef kitlesini net anlatir.
- "Pusula" karar verme ve rehberlik hissi verir.
- Affiliate, danismanlik, premium rehber ve sponsorlu icerik modellerine uygundur.
- Ucretsiz GitHub Pages URL'si de okunabilir: `kobipusulasi`.

## Ucretsiz Alternatifler

- GitHub Pages: `schillereu.github.io/kobipusulasi`
- Netlify: `kobipusulasi.netlify.app`
- Vercel: `kobipusulasi.vercel.app`
- Cloudflare Pages: `kobipusulasi.pages.dev`

GitHub Pages su an en pratik ve portfoy dostu secenektir.

## Ileride Ucretli Domain Alinirsa

Tercih edilebilecek domainler:

- `kobipusulasi.com`
- `kobiteknolojirehberi.com`
- `dijitalkobirehberi.com`
- `teknolojidunyasi.com`

Domain satin alindiktan sonra `scripts/set-custom-domain.ps1` ile GitHub Pages custom domain ayari yapilabilir.
